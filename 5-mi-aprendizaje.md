# COMPLETAR  
Comparando sus conocimientos antes de hacer la práctica con sus conocimientos después de hacer la tarea, explicar los principales aprendizajes logrados para beneficio de su formación profesional.  
Si solucionó un problema presentado al realizar la práctica también se debe documentar.

De docker aprendi a crear contenedores, ejecutarlos y entrar dentro del contenedor para hacer configuraciones, crear redes y conectar contenedores con ellas. Ádemas aprendi a linkear contenedores para que funcionen como cliente servidor o registrar en la base de datos. Solucione los probles de las variables de entorno necesarias para cada contenedor mediante los logs que furon muy útiles.

Consultar: Cómo se gestionan datos confidenciales con los secretos de Docker (Docker Secrets).
 Es un mecanismo diseñado para gestionar información sensible (como contraseñas, claves API o certificados) de forma más segura que las tradicionales variables de entorno. A diferencia de estas últimas, los secretos se cifran tanto en reposo como en tránsito y no se guardan en las capas de la imagen ni se muestran en los logs del proceso.
